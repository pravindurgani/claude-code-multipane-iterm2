# M5 Max AI Workstation — Quick Reference Card
> **Hardware edge:** M5 Max Fusion Architecture delivers ~614GB/s bandwidth. MLX hits 200+ tok/s on 14B models. Use MLX for performance-critical tasks, Ollama for tooling compatibility.

## 🚀 Daily Startup
```bash
ollama serve &                    # Start local models
open -a Docker                    # Start containers
osascript -e '...'               # Open multipane iTerm2
```

---

## 🤖 Model Selection Cheatsheet

| Task | Command | Model |
|------|---------|-------|
| **Don't know / default** | `llm-smart "..." fast` | Qwen 3 14B |
| **Code generation** | `llm-smart "..." code` | Qwen 2.5 Coder 32B |
| **Reasoning/audit** | `llm-smart "..." reason` | DeepSeek-R1 32B |
| **Direct model access** | `llm-fast / llm-code / llm-reason` | Specific model |
| **Claude Code (role-aware)** | `cc` | Based on iTerm profile |

### 80/20 Default Rule
**Not sure which model?** Use `cc` (Claude Code) or `llm-fast` (Qwen 3 14B). Escalate only when: security review needed, >50 files, production pipeline stage, or API is down.

## ✅ Pre-Ship Gate Check
```bash
gate  # Runs: pytest → ruff → tsc → docker build
```

**8 Gates (from Optimal Build Process):**
1. Problem explainable without tech? → Think more
2. Architecture reviewed? → Review now
3. CI passes? → Fix CI first
4. Happy path works? → Fix pipeline
5. Scope matches Step 1? → Cut scope
6. HIGH/CRITICAL fixed? → Fix before ship
7. Alerts on failure? → Monitoring not live
8. Redeploy <5 min? → Automate more

## 💡 Key Principles

1. **Pipeline discipline** — Fixed stages, typed boundaries
2. **Trust but verify** — LLM output always validated
3. **Degrade gracefully** — Every feature has fallback
4. **Route by complexity** — Cheapest accurate model wins
5. **Local for privacy** — Client data never leaves machine

## 🔧 Troubleshooting

| Issue | Fix |
|-------|-----|
| Model won't load | `ollama stop <model>` to free memory |
| Claude Code hooks fail | `python3 -m json.tool ~/.claude/settings.json` |
| Slow image gen | Close apps, use Q4 models, lower res |
| Transcription errors | Use `--language en` flag |
